What is network traffic? Data is broken into packets — small units each carrying a source & destination IP address, protocol type, and payload. Each packet is forwarded hop-by-hop: the firewall inspects and filters it, the core router selects the best path, the distribution switch delivers it to the correct VLAN, and the destination host reassembles the data.
